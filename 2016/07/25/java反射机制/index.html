java反射

所有的类都是java.lang.Class类的对象
获得Class实例的三种方法： 

    Class c1 = Foo.class;
    Class c2 = foo1.getClass();
    Class c3 = Class.forName("完整类名");

可以通过c1,c2,c3创建Foo的实例 

Foo foo = (Foo)c1.newInstance();

用new创建对象是静态加载类，在编译时刻就要加载所有可能使用的类，缺点是有一个类有问题，整个程序就编译不通过。功能性的类最后使用动态加载类。需要强制类型转换时，可使功能性类实现同一个接口。

获取方法信息

获取类名称：c.getName()
获取所有方法： 

Method[] ms = c.getMethods();//c.getDeclaredMethods()
getMethods获取所有public方法的信息，getDeclaredMethods是获取该类自己声明的成员变量的信息 得到方法返回值类型的类类型： 
Class returnType = ms[i].getReturnType();
returnType.getName();
得到方法名称：`ms[i].getName();` 得到参数类型的类类型： 
Class[] paramTypes = ms[i].getParameterTypes();

获取成员变量：

Field[] fs = c.getFields();//c.getDeclaredFields();
得到成员变量类型的类类型: 
Class fieldType = fs[i].getType();
得到成员变量的名称: 
String fieldName = fs[i].getName();

获取构造方法：

Constructor[] cs = c.getConstructor();//c.getDeclaredConstructors();

方法的反射：

  • 获取方法 
    Method m = c.getMethod("print", new Class[]{int.class, int.class});
    或 
    Method m = c.getMethod("print",int.class,int.class);
  • 方法的反射
    Object o = m.invoke(a1,new Object[]{10,20});
    或
    Object o = m.invoke(a1,10,20);

    效果等同于a1.print(10,20);

其他

反射的操作都是编译之后的操作
java中集合的泛型，仅是防止错误输入的，只在编译阶段有效，绕过编译就无效了，可以通过方法的反射来验证。
